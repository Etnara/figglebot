# Figglebot
  
### Usage
Edit User Variables:
```
token = "Discord Bot Token"
channel_id = Channel ID
email = "Generation Esports Email"
password = "Generation Esports Password"
```

Run Via Command Line:
```
python bot.py
python bot.py <Seconds to Wait>
python bot.py 3600
```

### Requires:
- Python

### Dependencies:
- Discord
- Selenium
- Webdriver_Manager
